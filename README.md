# Fiery Slash Shader (Godot)

A dynamic **fiery slash VFX shader** for **Godot (3D / Spatial)** designed for action games, RPGs, and stylized combat systems.

This shader combines animated noise, glowing gradients, flicker control, UV distortion, and dissolve transitions to create a powerful, cinematic sword-slash or energy-cut effect.
